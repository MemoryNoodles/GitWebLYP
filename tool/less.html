less

css预处理器less

Less是一门CSS预处理语言，它扩充了CSS语言，增加了诸如变量、混合（mixin）、函数等功能，让CSS更易维护、方便制作主题、扩充。

安装

在服务器端最容易的安装方式就是通过 npm （node.js 的包管理器），方法如下：npm install -g less

webstrom配置

  • 1、打开webstorm的file->settings ->External Tools，点击左上角的“加号”标志add，进入对话框
  • 2、progress:找到node.js下的less.cmd文件；例如：D:\nodejs\lessc.cmd
  • 3、arguments:--no-color --plugin=less-plugin-clean-css $FileName$ $FileParentDir$\css\$FileNameWithoutExtension$.css --source-map;
  • 4、output:$FileNameWithoutExtension$.css:$FileNameWithoutExtension$.css.map;

使用

1、使用变量

  • 1、以@作为变量的起始标识，变量名由字母、数字、_和-组成
  • 2、定义时 "@变量名: 变量值;" 的形式；引用时采用 "@变量名" 或 "@{变量名}" 的形式；

2.嵌套(Nested)

3.父选择器引用(ParentSelector)

  • 采用&引用完整的父选择器
  • 可通过追加和预追加的方式加工&，从而生成新的选择器
  • 同一个选择器可使用多个&
  • 通过在选择器后添加 "空格&"的方式，可将当前选择器排列到最前面
  • &指向组选择器时，会生成新的组选择器

4.继承

5.混合(Mixin)

在2016前各个浏览器还保留有私有属性的时候，运用混合无疑是无敌的。

6.运算符

Less还支持+、-、*、/运算符。但对单位不一致的运算数进行运算要注意以下两点

  • 1、运算数与运算符间必须用空格分隔；
  • 2、以第一个运算数的单位作为运算结果的单位；

7.函数

其他的函数请参考官网。

8.导入指令(Import)

less样式文件可通过@import '文件路径'; 引入外部的less文件。

  • 1、不带扩展名或带.less的扩展名均被视为less文件;
  • 2、@import可出现在任何位置，而不像css的@import那样只能放在文件第一行。
