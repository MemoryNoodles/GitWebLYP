express-generator

express应用生成器

第一步：全局安装 npm install express-generator -g

第二步：新建项目 express myProject

第三步：根据提示在命令行写入cd blog && npm install进行安装package.json下的依赖。

第四步：根据提示在命令行写入DEBUG=myapp:* & npm start

工程目录的结构说明

  • public：存放 image、css、js 等文件
  • routes：存放路由文件
  • views：存放视图文件或者说模版文件
  • bin：存放可执行文件
