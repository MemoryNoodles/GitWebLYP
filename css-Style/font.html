字体的设置

font复合属性：

该属性用于设置网页中文本的基本显示样式，如字体类型、字体粗细、字体大小、字体系列，甚至还可以直接通过对“font”属性多值的设置实现对文本行高的设置。

font-style:用于设置字体类型，可设置以下值：
normal：普通字体
oblique：倾斜字体
italic：斜体
font-weight：用于设置字体粗细，可设置以下值：
normal：正常粗细
bold：粗体
bolder：更粗的字体
lighter：更细的字体
font-size：用于设置字体大小，可设置以下值
px（像素）
em（em 的计算是基于父级元素的）
rem（基于根元素html）
%（相对于浏览器默认字体大小的百分比）
font-family：用于设置页面所显示的是什么字体，
“宋体”、“微软雅黑”、“黑体”、“Arial”、“Helvetica”、“Tahoma”、“Verdana”、“Lucida Grande”
