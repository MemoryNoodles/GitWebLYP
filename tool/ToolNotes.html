安装nodejs

进入官网下载node安装包

安装后修改全局安装时的默认路径，不修改就会默认安装在c盘

  • 1、打开'nodejs安装目录/node_modules/npm/.npmrc'，修改此文件为：prefix = '自己想安装的文件夹路径'
  • 2、修改prefix的值：npm config set prefix "自己想安装的文件夹路径"

确认是否修改正确：在命令行输入npm config ls

由于npm是老外的东西，所以直接安装会很慢我们采用淘宝镜像，安装过后只需要在所有的命令加c就行了。

npm install -g cnpm --registry=https://registry.npm.taobao.org

使用命令行

  • 1、说明：什么是命令行？命令行在OSX是终端（Terminal），在windows是命令提示符（Command Prompt）；
  • 2、打开命令提示符执行下列命令（打开方式：window + r 输入cmd回车）
  • 3、cd定位到目录，用法：cd + 路径；(快速切换盘符并更换到相应目录怎么办？？我教你，哈哈哈)
  • 4、cls清空命令提示符窗口内容。

npm介绍

  • 1、npm（node package manager）nodejs的包管理器，用于node插件管理（包括安装、卸载、管理依赖等）；
  • 2、使用npm安装插件：命令提示符执行npm install <name> [-g] [--save-dev]
  • 3、使用npm卸载插件：npm uninstall <name> [-g] [--save-dev]删除全部插件借助rimraf，先全局安装然后使用rimraf node_modules
  • 4、使用npm更新插件：npm update <name> [-g] [--save-dev]
